# AI-LEAD-QUALIFIER
This prototype demonstrates AI-assisted lead qualification, API integration, webhook-style automation, and spreadsheet logging using free or free-tier tools suitable for a small business workflow.

## Features
- Lead qualification form
- Flask backend API
- Frontend validation
- JSON request/response handling

## Running the Project
### Backend
```bash
cd backend
python app.py
```
### Frontend
Open `frontend/index.html` in a browser.
